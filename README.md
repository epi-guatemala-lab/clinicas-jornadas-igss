# Clínicas de Empresa & SIPRESALUD — IGSS

Portal interno para gestión de jornadas, inauguraciones de clínicas de empresa,
talleres y cálculo de costos del Departamento de Medicina Preventiva,
Subgerencia de Prestaciones en Salud, IGSS Guatemala.

**Sitio público**: https://epi-guatemala-lab.github.io/clinicas-jornadas-igss/

Stack: React 18 + Vite + Tailwind. Auth y datos vía API REST (backend privado).
